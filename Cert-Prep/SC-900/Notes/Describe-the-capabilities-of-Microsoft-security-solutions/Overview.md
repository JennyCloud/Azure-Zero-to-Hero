# üß© How It All Connects

Microsoft security solutions are:

- AI-driven
- Cloud-native
- Built around Zero Trust principles

They share signals across products.

Example:
- Defender detects suspicious login
- Entra ID flags risky user
- Sentinel correlates activity
- Conditional Access blocks session

This is called **XDR (Extended Detection and Response)**.
